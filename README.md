# Postman API Test Project

This project contains API test scenarios created using Postman.

## 🔹 Technologies
- Postman
- JavaScript (Postman test scripts)
- REST API

## 🔹 Test Scenarios
- Status code validation
- Response time validation
- Schema validation
- Field validation
- Negative scenarios

## 🔹 How to Run
1. Import the collection into Postman
2. Set environment variables
3. Run using Collection Runner
